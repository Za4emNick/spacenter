# Zen Spa Center — cleaned dark website

Static website with two pages:

- `index.html` — main page: spa info, locations, advantages, gallery
- `services.html` — services page: compact service menu with modals

Files:

- `styles.css` — black luxury / champagne gold visual system
- `data.js` — hotels, gallery, services and translations
- `app.js` — language switching, modals, gallery and WhatsApp booking
- `logo.png`, `Massage.png`, `Turkishbath.png`, `Face.png`, `Special.png` — existing visual assets

Notes:

- Language selection appears on first visit and is saved in `localStorage`.
- WhatsApp booking button is fixed at the bottom on both pages.
- WhatsApp opens only after selecting a hotel.
- If booking from a service modal, the WhatsApp message includes service + hotel.
